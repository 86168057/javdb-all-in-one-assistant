# JAVDB全能助手 - 使用说明

> 一个功能强大的 JAVDB 增强脚本，集成 EMBY 服务器联动、多站点搜索、预览图查看、磁力链管理等功能

## ✨ 核心功能

### 🔗 EMBY 服务器联动
- **实时入库状态显示**：详情页和列表页自动显示媒体是否已入库 EMBY
- **一键跳转**：点击"已入库"标签直接跳转到 EMBY 服务器中的媒体详情页
- **智能同步**：支持多服务器配置，自动全量同步媒体库并建立本地索引
- **实时感知**：当您在 EMBY 中新增或删除媒体时，页面状态会实时同步更新

### 🔍 多站点搜索
详情页一键搜索当前番号到多个平台：
- 🔥 98堂论坛
- 🌐 BTSOW BT搜索
- 📚 JAVDB 站内搜索
- 🎬 JAVBUS
- 🔎 Google 搜索

### 🖼️ 预览图查看
- **详情页预览**：一键加载并查看所有高清预览图
- **大图查看器**：点击预览图进入全屏查看模式，支持缩放、滚轮切换
- **智能加载**：仅显示大图，过滤缩略图

### 🧲 磁力链管理
- **实时显示**：磁力链按钮角标实时显示可用数量
- **智能排序**：有字幕的资源自动排在最前面
- **详细信息**：显示文件大小、日期、标签（字幕/高清）
- **一键复制**：点击即可复制磁力链接

### 🏆 排行榜快捷访问
- **全局浮动按钮**：页面右下角固定显示排行榜入口（🏆）
- **多种排行榜**：影片、演员、有码、无码、欧美、年度排行榜
- **VIP提示**：清晰标注免费和VIP排行榜

### ⚙️ 便捷设置
- **图形化配置**：右下角设置按钮，可视化管理 EMBY 服务器
- **连接测试**：添加服务器时自动测试连接（3秒超时保护）
- **配置备份**：支持导出/导入配置文件，数据永不丢失

### 2. 安装脚本
点击 GreasyFork 页面上的"安装此脚本"按钮，Tampermonkey 会自动识别并安装。

## 🚀 快速开始

### 第一步：配置 EMBY 服务器（可选）

如果您希望使用入库状态显示功能：

1. **访问 JAVDB 任意页面**
2. **点击右下角的设置按钮**（⚙️）
3. **点击"添加 emby 服务器"**
4. **填写信息**：
   - **服务器名称**：例如 "家庭影院"（默认为 emby）
   - **服务器地址**：例如 `http://192.168.1.100:8096`
   - **API Key**：登录 EMBY 后台 → 设置 → 高级 → API 密钥 → 新建

5. **点击"连接"按钮测试**
   - 连接成功会显示绿色标签"在线已连接"
   - 失败会显示具体错误信息（地址错误/API Key错误/超时等）

6. **点击"保存并同步"**
   - 脚本会全量抓取您的媒体库（可能需要几秒到几分钟）
   - 同步完成后，JAVDB 页面会自动显示入库状态

### 第二步：享受功能

配置完成后，您会在 JAVDB 页面看到：

**详情页**：
- 番号标题下方显示多站点搜索按钮
- 番号上方显示 EMBY 入库状态（已入库/未入库/未添加服务器）
- 右下角有两个浮动按钮：⚙️设置、🏆排行榜

**列表页**：
- 每个视频卡片显示入库状态角标
- 卡片下方有预览图、磁力链按钮

## 🔧 常见问题

### Q1: 为什么 EMBY 标签不显示？
**A**: 请检查：
1. 是否已添加并连接 EMBY 服务器
2. 服务器地址是否正确（需要包含 `http://` 或 `https://`）
3. API Key 是否有效
4. 点击设置中的"保存并同步"按钮，确保已同步媒体库
5. 打开浏览器控制台（F12）查看是否有错误提示

### Q2: 同步很慢怎么办？
**A**: 
- 首次同步需要抓取所有媒体，可能需要几分钟（取决于媒体数量）
- 后续会自动增量同步，速度很快
- 如果超过10秒未响应，会显示"连接超时"

### Q3: 磁力链按钮显示"0"？
**A**: 
- 这表示该番号在 JAVDB 上没有可用磁力链接
- 不是脚本问题，而是该资源本身没有磁力

### Q4: 排行榜显示"需要VIP会员"？
**A**: 
- JAVDB 的大部分排行榜需要 VIP 才能访问
- 只有"演员 TOP250"是免费的
- 脚本已用 🔒 图标标注付费排行榜

### Q5: 搜索按钮跳转到98堂但没有自动搜索？
**A**: 
- 98堂有时需要手动点击"满18岁"确认
- 脚本会尝试自动点击，但可能因页面变化失效
- 手动点击一次即可

### Q6: 脚本加载很慢或不生效？
**A**: 
- **切勿**通过拖入浏览器的方式安装脚本
- 必须通过 Tampermonkey 的"安装脚本"功能安装
- 刷新页面，等待1-2秒即可看到效果
